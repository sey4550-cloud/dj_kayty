# DJ Kayty - Official Website

Welcome to the official website of DJ Kayty. This is a modern, responsive DJ portfolio featuring live sets, booking information, courses, and more.

## 📋 Features

- **Home**: Hero section with latest updates via ticker
- **Mixtapes**: Stream and download exclusive audio sets
- **Booking**: Private events, club nights, and wedding bookings
- **Courses**: Beginner, intermediate, and master DJ training programs
- **Gallery**: Photo portfolio showcase
- **Subscribe**: Newsletter signup for updates
- **Mobile-Friendly**: Fully responsive design for all devices

## 📁 Project Structure

```
dj kayty/
├── index.html           # Home page

├── subscribe.html       # Newsletter signup

    ├── intermediate/
    └── master/
```

## 🎨 Design

- **Color Scheme**: Blue (#0A369D) & Gold (#FFD700)
- **Typography**: Bebas Neue (headings) & Inter (body)
- **Icons**: Font Awesome 6.4.0

## 📱 Responsive Design

- Desktop-first approach
- Breakpoints at 800px, 640px, and 480px
- Mobile hamburger menu support
- Optimized video and image scaling

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required—pure HTML, CSS, and vanilla JavaScript
